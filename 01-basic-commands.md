# Linux Basic Commands

## Navigation

```bash
pwd        # Show current directory
ls         # List files
ls -la     # List all files with details
cd folder  # Change directory
cd ..      # Go back one directory
```

## File Operations

```bash
touch file.txt
mkdir project
cp file1 file2
mv old.txt new.txt
rm file.txt
rm -r folder
```

## Viewing Files

```bash
cat file.txt
less file.txt
head file.txt
tail file.txt
```

## Help

```bash
man ls
history
clear
```
